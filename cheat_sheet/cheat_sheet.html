Semantic HTML Cheat Sheet

What is Sematic HTML?

Semantic HTML is code that gives meaning to the code written. It allows developers and the browser to know more specifically what content the HTML contains.

Semantic HTML tag examples

  • header

    • Denotes the content at the top of the page

  • nav

    • Denotes the navigational links

  • main

    • Denotes the content between header and footer area

  • footer

    • Denotes the content at the body of the page

  • video

    • Denotes a video implementation

  • audio

    • Denotes audio implementation

  • embed

    • Denotes any type of implementation

  • figure

    • Denotes a media element is being contained

  • figurecaption

    • Denotes the figure element description

  • section

    • Denotes am area of the document

  • article

    • Denotes content that stands independent of the rest of the document

  • aside

    • Denotes additional information that can enhance another element but is not required to understand the main content
